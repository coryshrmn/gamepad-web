Event-based Gamepad Web Backend

GitHub repository

This crate extends the Gamepad Web API, to emit events for button and axis state changes.

Our goal is to resemble a native API, which emits an event for each state change, unlike the Web API, which only allows state querying.

Quick Start

use gamepad_web::*;

// start listening for Gamepad events
let mut monitor = Monitor::new();

fn update() {
    // process new input events
    while let Some(event) = monitor.poll_mapped() {
        match event {
            MappedEvent::ButtonPress(Button::South) => // "A" on Xbox
                player.jump(),
            MappedEvent::Axis(Axis::LeftStickX, x) =>
                player.set_velocity(x),
            _ => ()
        }
    }
}
